# Contorch
A lightweight, zero-dependency container orchestrator and monitoring dashboard.  
Run one container and get a full web UI to manage all other containers on your host.

Contorch is designed for developers and small teams who want simple, powerful container management without Kubernetes, Swarm, or complex tooling.

---

## Features

### Container Dashboard
- List all containers on the host  
- View status, health, uptime, image, and configuration  
- Inspect environment variables and runtime settings  

### One-Click Actions
- Start, stop, restart containers  
- View container logs (live stream and history)  
- Remove containers and images safely  

### Disk Usage Explorer
- See which containers, images, and volumes use the most disk space  
- Identify unused or oversized data  
- Clean up safely from the UI  

### System Overview
- Host CPU, memory, and disk usage  
- Docker engine status  
- Build cache size and reclaimable space  

### Zero Installation
- No agents  
- No dependencies  
- No YAML  
- Everything runs inside a single Contorch container
